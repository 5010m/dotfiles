# How to use

To use this dotfiles. try to do below steps.

## Step 1. Clone Repository

<pre>
$ cd ~
$ git clone https://github.com/5010m/dotfiles.git 
</pre>

## Step 2. Deploy dotfiles and plugins

<pre>
$ cd dotfiles
$ ./dotfiles deploy
</pre>
